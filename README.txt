# Simple Animal Sim (Pygame)

Tiny toy simulation with plants and animals eating, and reproducing.
Can be quite mezmerizing to look at.

Plants:
- Gain energy from "sun"
- Reproduce when energy is high

Animals:
- Seek nearest plant
- Eat to gain energy
- Reproduce when energy is high
- Die on starvation and occasional collisions (population control)

# How to run
pip install pygame
python main.py

There are som editable variables at the top of the file to play with
